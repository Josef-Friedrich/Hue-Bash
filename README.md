# Requirements
* [curl](curl)
* [jq](http://stedolan.github.io/jq/)

# Commands

## hue
------

### hue help

```
hue help
```
Show help

### hue set

```
hue set <lamp> <json>
```
```<lamp>```: Lamp number or 'all'.
### hue get

```
hue get <lamp> 
```
```<lamp>```: Lamp number or 'all'.
### hue ping

```
hue ping <lamp>
```
```<lamp>```: Lamp number or 'all'.

### hue stop

```
hue stop
```

### hue debug

```
hue debug
```

## huectrl
----------

```
huectrl start

```
