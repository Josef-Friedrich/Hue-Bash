########################################################################
# hueload-bridge
########################################################################

USAGE
	hueload-bridge <options>

OPTIONS

	-h, --help
		Show this help message.

DESCRIPTION
	'hueload-bridge' loads the default scene from the DEFAULT_SCENE
	variable in the config file when the Hue bridge is online.

EXAMPLES
	hueload-bridge
	hueload-bridge --help
